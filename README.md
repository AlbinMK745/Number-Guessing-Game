# Number Guessing Game

A fun interactive number guessing game where players try to guess a random number between 1 and 100.

## Features
- Enter your name and start playing
- Get feedback on your guesses (higher/lower)
- Track your attempts
- Leaderboard scoreboard
- Animated UI with particle effects

## Files
- `index.html` - Main game interface
- `game.js` - Game logic
- `style.css` - Styling

## How to Play
1. Open `index.html` in your browser
2. Enter your name and lock it
3. Guess a number between 1-100
4. Follow the feedback hints
5. Submit your final guess to see your score
